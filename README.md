# 🧠 Quiz Application

A simple and interactive Quiz Application built to test users' knowledge with multiple-choice questions. This project is perfect for beginners to understand basic programming concepts like logic building, user interaction, and data handling.

---

## 🚀 Features

* Multiple-choice questions (MCQs)
* Score calculation
* Instant feedback on answers

* Responsive and user-friendly interface
* Restart quiz option

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
  

---

## 📂 Project Structure

```
quiz-app/
│── index.html
│── style.css
│── script.js
│── questions.js
│── README.md
```



## 🎯 How It Works

* The app displays one question at a time
* User selects an answer
* The system checks if it is correct
* Score is updated accordingly
* Final score is shown at the end
